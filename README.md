# install guide
You can use pip to install all package yourself, just ignore below
```
// install pipenv
$ pip install pipenv

//install dependency
$ pipenv install

//to start environement in console
$ pipenv shell

// run server
python manage.py runserver

//with port
python manage.py runserver 8080
```
# use .env for email, tehn runserver
//in terminal
$ source .env
```
